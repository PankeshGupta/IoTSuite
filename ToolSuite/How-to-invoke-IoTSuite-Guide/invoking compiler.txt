Step 1 : Extract Template.rar file to a specific Path 
(for instance : D:\Template)

Step 2: Run ToolSuite with the following configuration
(Run Configurations)

Conditions:
   D:/Template/CodeForDeployment folder must be empty.
   D:/Template/Device Driver must have two folders: (1) Android (2) JavaSE
   D:/Template/JavaSEDeviceDriver folder should have project, but "src" folder with "iotsuite"
   D:/Template/AndroidDeviceDriver folder should have project, but "src" folder with "iotsuite"
   D:/Template/ApplicationLogic folder should have project, but "src" folder with "iotsuite"

For generating device drivers
==============================
D:\Spec\vocab.txt
D:\Spec\arch.txt
D:\Spec\deply.txt
D:\ToolSuite\IoTSuite\ToolSuite\StringTemplateFiles\
dd
D:\Template\

Write code into  "DeviceImpl" code.

Output:

This stage produce code in the following directory
 >>  D:/Template/JavaSEDeviceDriver/src: (a) deviceImpl (b) factory  (c) framework (d) logic
 
This stage produce code in the following directory 
(The generation have to fix into compiler. The current version does not 
generate code into this directory right now)
 >>  D:/Template/AndroidDeviceDriver/src: (a) deviceImpl (b) factory  (c) framework (d) logic


For generating architecture framework
=====================================
D:\Spec\vocab.txt
D:\Spec\arch.txt
D:\Spec\deply.txt
D:\ToolSuite\IoTSuite\ToolSuite\StringTemplateFiles\
af
D:\Template\

Write code into  "Application Logic" code.

Output: 
This stage produces code in the following directory
	>> D:/Template/ApplicationLogic/src: (a) logic  (b) framework
	
The application developer have to write logic
